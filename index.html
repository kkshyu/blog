import-cra

終於試成功了！
在一個 CRA 專案另一個 CRA 專案～～
以後就可以共用核心的 CRA
然後把擴增的路由、主題與頁面放進去延伸
這樣子新的客戶要怎麼客製化都不怕了！
具體作法是把被引用的 CRA 先編譯好上 NPM 或 git
然後另一個直接 install 下來
這樣就可以像是以下使用了 App 等級的 component 了

third-party-cookie

今天終於徹底了解了 Chrome 對第三方 cookie 的限制
我們根本不會受到影響rrrrr
不需要使用 proxy 轉成第一方
只需要把 Cookie 設定成 SameSite: none; Secure 即可＝＝
