2020-09-03
cloudflare SSL mode issue

太刺激啦～～
使用 Flexible SSL Mode 則不能設定 cookie 唷！
Cloudflare 會把你的 cookie 給移除掉
只會留下預設的 __cfduiud 這個 cookie
所以如果使用 Cloudflare 又需要設定 cookie 的話
記得使用 Full mode 或是 Full(Strict) mode ～
